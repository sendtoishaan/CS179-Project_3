Description: Given different types of starting positions(states) for crates on a ship, make the ship legally balanced(weight wise) by moving containers/crates from one place to another through a series of moves optimally(least amount or number of moves).

Programming Language: Python

3. collections
4. tkinter for GUI GRID VISUALIZATION
